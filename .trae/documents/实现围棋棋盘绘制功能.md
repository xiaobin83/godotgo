# 实现围棋棋盘绘制功能（支持手动设置棋盘大小）

## 1. 创建 go_board.gd 脚本
- 解析 `showboard` 命令返回的棋盘信息
- 支持手动设置棋盘大小（提供 set_board_size 函数）
- 存储棋盘的结构化数据，包括：
  - 棋盘大小
  - 每个交叉点的状态（空、黑子、白子、星位）
  - 双方的俘虏数
- 提供访问棋盘数据的方法

## 2. 创建 go_board_simple_renderer.gd 脚本
- 读取 `go_board` 中的棋盘信息
- 使用 `ImmediateMesh2D` 绘制棋盘，包括：
  - 棋盘网格（适应不同大小）
  - 星位（根据棋盘大小自动计算）
  - 黑白棋子
  - 俘虏数信息
  - 棋盘坐标标记（字母和数字）
- 支持缩放和定位

## 3. 修改 main_ui.gd
- 集成 `go_board` 和 `go_board_simple_renderer`
- 当收到 `showboard` 命令的响应时，更新 `go_board` 中的数据
- 触发棋盘重绘

## 4. 更新场景文件
- 在 main.tscn 中添加 `go_board` 和 `go_board_simple_renderer` 节点
- 调整节点布局，确保棋盘正确显示

## 5. 测试功能
- 连接到服务器
- 发送 `showboard` 命令
- 验证棋盘是否正确绘制
- 测试不同棋盘状态的显示效果
- 测试通过函数设置不同棋盘大小的效果
